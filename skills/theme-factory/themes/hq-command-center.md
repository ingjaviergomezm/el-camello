# HQ Command Center

Un tema inspirado en la sala de control de un edificio corporativo de alta tecnología. Acero y concreto como estructura. Neón verde como el pulso vital que indica que todos los sistemas están en línea. Ámbar como las alertas y los acentos que capturan la atención.

## Color Palette

- **Gunmetal Steel**: `#2D3436` - Fondos principales, la estructura del edificio
- **Neon Pulse**: `#00E676` - Acento primario, el "sistemas en línea" del HQ
- **Amber Alert**: `#FFB300` - Acentos secundarios, llamados a la acción, highlights
- **Clean White**: `#FAFAFA` - Texto sobre fondos oscuros, aire y claridad
- **Slate Gray**: `#636E72` - Texto secundario, subtítulos, elementos discretos

## Typography

- **Headers**: Space Grotesk Bold (geométrica, arquitectónica, moderna)
- **Body Text**: IBM Plex Sans (técnica pero legible, toque industrial)

## Best Used For

Showcases técnicos, dashboards de operaciones, presentaciones de infraestructura IA, pitch decks de producto con identidad industrial-moderna.
